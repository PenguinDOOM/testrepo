# testrepo

これは動作確認とか、ちょっとした試作をするときのテスト用リポジトリです。

まだ中身はほとんどありませんが、ここからメモを置いたり、サンプルコードを足したり、実験用の設定を試したりできます。

## 何に使うか

- 小さなコード断片の動作確認
- 設定ファイルやツールの試験導入
- 一時的なメモや作業ログの置き場

## 最初の一歩

例えば、次のようなものから追加していくと使いやすくなります。

- `notes/` に検証メモを置く
- `samples/` にサンプルコードを置く
- 検証手順をこの README に追記する

## 置き方の目安

- すぐ消してよい試作は、用途が分かる名前のファイルで直置きする
- 後で見返す可能性があるものは、フォルダを切ってまとめる
- 試した理由や結果は、コードだけでなく短いメモも残す

## メモの書き方例

試作や確認をしたときは、次のような観点を残しておくと再利用しやすくなります。

- 何を試したか
- どのコマンドや設定を使ったか
- 結果がどうだったか
- 次に何を試すか

## 補足

雑に使ってよい場所ですが、あとから自分で見返して分かる程度には整理しておくと便利です。
